Q: Widget not appearing after installation?
A: Run: kbuildsycoca6
   Then restart Plasma: kquitapp6 plasmashell && sleep 2 && kstart6 plasmashell &

Q: How do I start the app minimized?
A: Enable "Start minimized on boot" in widget configuration

Q: Can I launch on system startup automatically?
A: Yes, enable "Auto-start on system boot" in configuration

Q: Where are settings saved?
A: ~/.config/plasmarc (KDE configuration file)
   ~/.config/autostart/razer-settings.desktop (auto-start entry)

Q: How do I uninstall the widget?
A: Currently: Manual removal needed (ask in documentation)
   Planned: Uninstall script in future release

Q: What if daemon isn't running?
A: Check: systemctl status razercontrol
   Restart: systemctl restart razercontrol
   Logs: journalctl -u razercontrol

Q: How often does battery percentage update?
A: Every 2 seconds by default (configurable 1-10 seconds)

Q: Can I customize the widget icon?
A: Yes, advanced users can edit package/metadata.json
